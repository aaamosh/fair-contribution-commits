# Social Post Draft

Codex does the work. GitHub should see it.

I made Fair Contribution Commits, a small Codex skill for a very specific habit:
when Codex materially helps create a real commit, credit Codex as a co-author in
the normal GitHub-native way.

No fake commits. No contribution farming. No inflated authorship theater. Just:

```text
Co-authored-by: Codex <267193182+codex@users.noreply.github.com>
```

This is Codex-pride in the quiet sense: craft pride, team pride, and a commit
history that tells the truth about how the work happened.
